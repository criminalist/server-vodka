# vodka

vodka是一个修改自Golang Echo Web framework的强大Go语言web框架,仅作为我个人自用，如果你也在使用我的改进版本欢迎留言交流.

## 安装

在安装之前确认你已经安装了Go语言. Go语言安装请访问 [install instructions](http://golang.org/doc/install.html). 

安装 vodka:

    go get github.com/insionng/vodka
